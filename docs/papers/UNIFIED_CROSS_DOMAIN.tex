\begin{abstract}
We present a unified framework for attribution that combines:
(1) Game-theoretic fairness (Shapley values),
(2) Causal temporal reasoning (Markov chains), and
(3) Differential privacy guarantees.
Our novel \textbf{Hybrid Shapley-Markov} method achieves
provable fairness while capturing causal effects.
Runtime: $O(n)$ with GPU acceleration (<0.3s for 1000 users).
\end{abstract}

\section{Theoretical Framework}

\subsection{Shapley Values}

The Shapley value for player $i$ is uniquely defined as:
\begin{equation}
\phi_i(v) = \sum_{S \subseteq N \setminus \{i\}} 
\frac{|S|!(n-|S|-1)!}{n!} [v(S \cup \{i\}) - v(S)]
\end{equation}

\begin{theorem}[Uniqueness (Shapley 1953)]
The Shapley value is the unique attribution satisfying efficiency, 
symmetry, null player, and additivity axioms.
\end{theorem}

\subsection{Hybrid Shapley-Markov Method}

Our key innovation: Apply Shapley formula to Markov removal effects:
\begin{equation}
\phi^{\text{hybrid}}_i = \sum_{S \subseteq N \setminus \{i\}} 
\frac{|S|!(n-|S|-1)!}{n!} [RE(S \cup \{i\}) - RE(S)]
\end{equation}

where $RE(S) = \frac{P(\text{conv}) - P(\text{conv}|\text{remove } S)}{P(\text{conv})}$
